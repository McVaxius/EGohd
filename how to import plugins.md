# How To Import Eyes of Gohd

## Dev Plugin Path

`Z:\EGohd\EGohd\bin\x64\Debug\EGohd.dll`

## Add The Plugin In XIVLauncher

1. Launch FFXIV through XIVLauncher.
2. Open Dalamud settings with `/xlsettings`.
3. Go to `Experimental`.
4. Add `Z:\EGohd\EGohd\bin\x64\Debug\EGohd.dll` to `Dev Plugin Locations`.
5. Open the plugin installer with `/xlplugins`.
6. Go to `Dev Tools > Installed Dev Plugins`.
7. Enable `Eyes of Gohd`.
8. Run `/egohd` in game to open the main window.
9. Use `/egohd config` if you want the settings window directly.
